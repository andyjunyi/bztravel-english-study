關係代名詞 which 的非限定用法 ▾
非限定關係子句用逗號隔開，which 指代前面的整個名詞或概念，提供額外資訊而非必要資訊。
This neighborhood offers plenty of solid food options and a walkable street, making it a practical base for exploring the day.
→ 'making it a practical base' = which makes it a practical base → 'which' 指代前面整個情況
[主句], which + V... / [主句], V-ing... → 非限定關係子句
Berlin has excellent public transport, ______ (這使得) easy to get around without a car.
which makes it
S + V + Adj + enough + to + V 的變化形式 ▾
'Good for anywhere' 是 'good enough to be used anywhere' 的簡化版，形容詞直接接 'for + 名詞'。
It's good for anywhere in the world.
→ good for = suitable for → 介系詞 for 直接接 anywhere，表達適用的範圍
S + be + Adj + for + [任意範圍]
This jacket is ______ (適合任何天氣) in spring.
good for any weather
