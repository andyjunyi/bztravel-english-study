the + 比較級, the + 比較級 — 「越...就越...」 ▾
這個句型用於表達兩個事物同步變化。結構為 'The + 比較級 + 主詞 + 動詞, the + 比較級 + 主詞 + 動詞'。
The farther south we fly, leaving the Okavango Delta behind, the drier everything becomes.
→ 'The farther...the drier' = 越往南飛...就越乾燥 → 第一個 the + 比較級是條件，第二個是結果
The + 比較級 + S + V, the + 比較級 + S + V → "越...就越..."
The closer we get to the Kalahari, the drier everything becomes.
→ 同樣的 'the + 比較級...the + 比較級' 結構 → 強調漸進的變化過程
The closer...the drier → "越接近...就越乾燥"
The more you practice, ______ (你就學得越好).
the better you learn
否定詞 + 比較級 — 強調最高級 ▾
使用否定詞（如 never, nothing, no）加上比較級，可以傳達強烈的最高級意味，比直接使用最高級更有感染力。
Never in my life have I seen zebras in such numbers.
→ 'Never...have I seen' = 我這輩子從未見過 → 否定副詞開頭引起倒裝：Never + V + S
Never + have/has + S + p.p. + ... → "從未..."（倒裝句）
Never ______ (我從未吃過) such delicious food.
have I eaten
