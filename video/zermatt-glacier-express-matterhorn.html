比較級疊加 — each + 比較級 + than the last (一個比一個更...) ▾
用 'each + 比較級 + than the last' 來描述一系列事物逐次遞增的程度，是一種優雅的英文修辭結構。
To reach the Klein Matterhorn, we switch through three lifts. Each one steeper than the last.
→ Each one steeper than the last = 第三條比第二條陡，第二條比第一條陡 → 強調逐漸增加的變化
Each + N + 比較級 + than the last → 「一個比一個更...」
We visited three museums in one day. ______ (一個比一個更令人印象深刻).
Each one more impressive than the last
S + V + 名詞片語 (同位語補充說明) ▾
用名詞片語放在句尾補充說明主詞或受詞的身份或性質。同位語（appositive）是一個名詞或名詞片語放在另一個名詞旁邊，用來補充說明它的身份。
The Matterhorn — that mountain that's on that Toblerone candy bar.
→ 'that mountain...' 是 The Matterhorn 的同位語 → 補充說明馬特洪峰的身份
[名詞] — [同位語補充說明]
Zermatt — ______ (那個禁止車輛通行的阿爾卑斯小鎮) — is a paradise for nature lovers.
that car-free alpine town
